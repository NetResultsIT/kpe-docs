Detailed Description

The KpeLogConfig class is the object that contains information about where and how KPE logs.

The log rotation params of this config also apply to underlying CTI and Voip engines, which will emit their specific log files in the path indicated in this config. Expected log files:

  • one for KPE
  • one for CTI engine
  • six for Voip engine

KpeLogConfig::KpeLogConfig ( QString path = QDir::currentPath())

KpeLogConfig creates a KPE log configuration object to be used when creating a new KpeQt object.

Parameters
paththe path where logs should be stored, defaults to the application directory

By default the log configuration object is configured as follows:

  • Logs are located where the application executable lives
  • No time or size rotation is configured
  • The log file is named: "kpelog.txt"
