Detailed Description

The IncomingCallDestinationDetails class represents an incoming call details. Implements JsonSerializable interface. The Json representation of the object is the following:

{
"calltype": "...",
"name": "...",
}

where:

  • callType is an IncomingCallType. Possible values are: direct, queue, group, callback (see IncomingCallType to full list of types).
  • name indicates the name of the caller (a queue name in case of a call from queue, a group name in case of a call from a group, an exten in case of direct call...)
